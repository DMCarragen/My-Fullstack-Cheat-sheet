Links e Imagenes en Html

Links

Los Links en html se usan con el tag a href="link interno o externo"> el nombre del botón del link y despúes se vuelve a cerrar con un tag a

el link tiene que tener un path ya sea externo con un url que comience con su debido protocolo ejemplo "https"

o en el caso de ser un link interno se escribe el path desde la pagina en la que esta el link hacia la pagina a la que se dirige, se utiliza ../ para utilizar el directorio root como referencia.

Imagenes

Para poner una imagen sin mucha especificacion se utiliza el tag img (Este Tag no se cierra)

dentro del mismo tag img se escribe src="path/imagen", y tambien el atributo alt que es un texto que describe la imagen.
